## 📖 Overview

**Gemini RAG Framework** is a production-ready Retrieval-Augmented Generation (RAG) system that answers questions from your documents using **Google Gemini AI**. It combines hybrid search (BM25 + Vector) with powerful LLM capabilities to provide accurate, source-attributed answers.

This project was built from the **Enterprise-RAG-Framework** and enhanced with **Gemini integration** for free, high-quality AI responses.

---

## ✨ Features

### 🔍 Hybrid Search
- **BM25 Keyword Search** - Exact keyword matching
- **Vector Search** - Semantic understanding with embeddings
- **Combined Results** - Best of both worlds

### 🧠 Gemini AI Integration
- **Gemini 2.5 Flash** - Fast, free AI responses
- **250 Requests/Day** - Generous free tier
- **Source Attribution** - Know where answers come from

### 📄 Document Processing
- **Multi-Format Support** - PDF, DOCX, TXT, HTML, Markdown
- **Smart Chunking** - Recursive, sliding window, semantic
- **Metadata Extraction** - Track document sources

### 🚀 Production Ready
- **Evaluation Suite** - Precision, recall, faithfulness metrics
- **Monitoring** - Track usage and performance
- **Security** - API key protection with .env
- **Interactive Q&A** - Real-time questioning
